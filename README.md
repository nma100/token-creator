# Token creator
Token Creator dapp allows you to create and deploy your own ERC-20 token.

The application integrates the UNS functionality "Login With Unstoppable"

## Build

Install dependencies :
```
npm install
```

Start development server :
```
npm start
```

Production build :
```
npm run build
```
